# SpringBoardProject_v1

## 사용 스킬
- **Java**
- **Jsp**
- **Spring**
- **JavaScript**
- **JQeury**
- **Mybatis**
- **MariaDB**

## 구현 기능

- **게시글 생성**
- **게시글 목록 보기**
- **게시글 수정**
- **게시글 삭제**
- **댓글 작성 및 보기**
- **댓글 수정**
- **댓글 삭제**
- **게시글 항목별로 검색 가능 (제목, 내용, 제목+내용, 작성자)**
- **게시글 조회수 기능**
- ( 회원가입/로그인 구현중... )
