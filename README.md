# SpringBoardProject_v1

## 사용 스킬
- **Java**
- **Jsp**
- **Spring**
- **JavaScript**
- **JQeury**
- **Mybatis**
- **MariaDB**

- **게시글 생성**
- **게시글 목록 보기**
- **게시글 수정**
- **게시글 삭제**
- **댓글 작성 및 보기**

- **게시글 항목별로 검색 가능 (제목, 내용, 제목+내용, 작성자)**
- **게시글 조회수 기능**
